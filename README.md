ViT-Starfield: Vision Transformer-Based Spacecraft Attitude Estimation
This repository contains the implementation of a Vision Transformer (ViT) framework for robust spacecraft attitude determination. The system integrates deep visual features with geometric polygon-angle embeddings to achieve high accuracy in "lost-in-space" scenarios, even under noise and occlusion.

📄 Abstract
Accurate attitude determination is essential for spacecraft navigation. Traditional star trackers often struggle under degraded imaging conditions. This framework fuses deep visual representations from a ViT with geometric descriptors of star polygons. The model is trained on synthetic data generated via Stellarium and optimized for real-time onboard inference.
